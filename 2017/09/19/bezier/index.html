贝塞尔曲线

2017-09-19

简介

贝塞尔曲线（Bezier Curve）是一种重要的数学工具，可以利用少量的点生成任意光滑的曲线。贝塞尔曲线最初由法国工程师皮埃尔·贝塞尔（Pierre Bezier）提出，他运用贝塞尔曲线设计汽车的主体部分。随后它被大规模用于设计领域中，例如Photoshop中的钢笔工具，还有动画中的缓动特效。

DEMO在这里，源码在GITHUB。

顺带一提，贝塞尔曲线和弗里德里希·威廉·贝塞尔(Friedrich Wilhelm Bessel)提出的贝塞尔函数（Bessel Functions）是不一样的，只是译名相同而已。

2阶贝塞尔曲线

给定一些点，即可生成一条贝塞尔曲线，这些点称为控制点。由3个控制点生成的贝塞尔曲线称为二阶贝塞尔曲线，如下图所示：

生成过程很简单，对于三个控制点A、B、C：

  1. 分别作线段AB，BC。
  2. 取{ t | 0 <= t <= 1 }，在AB取点D，使得AD / AB = t，同理在BC取点E。
  3. 连接DE，在DE中取F，使得DF / DE = t。
  4. 所有t值对应的F点集合即为贝塞尔曲线。

n阶贝塞尔曲线

贝塞尔曲线实际上可以取任意复数个控制点，由n + 1个控制点组成的贝塞尔曲线被称为n阶贝塞尔曲线。Photoshop中的钢笔和缓动动画中的缓动曲线都是3阶贝塞尔曲线。

作n阶贝塞尔曲线的方法是一个递归的过程：

  1. 对n+1个控制点连接出n条线段。
  2. 同2阶贝塞尔曲线，在n条线上取n个t分割点。
  3. 对于n个分割点，回到步骤1，直到分割点只剩1个，记为F点。
  4. 对于所有t，对应的F点集即为贝塞尔曲线。

可以看出，上述算法是个尾递归调用，DEMO中的蓝色贝塞尔曲线就是用递归法画出。n阶贝塞尔曲线的演示可以看这里。

贝塞尔曲线通式

n阶贝塞尔曲线可以写成t的参数通式方程：

B(t) = Σ(PTi C(n, i) (1 - t) ^ (n - i) * t ^ i)

通式可以减小递归调用时的切换消耗，DEMO中的红色贝塞尔曲线就是用通式法绘制的。但是实测中，由于V8引擎尾递归的高效率，递归绘制效率反而高于通式绘制，相对于通式绘制只消耗60%的时间。
